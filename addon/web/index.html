Discord Read-Only Bot

This panel is a zero-permission pointer to the external Discord bot package. The bot runs outside the console and talks only to the bearer-token protected Discord adapter API.

Runtime Boundary

  • No Docker socket mount.
  • No database credentials.
  • No raw console command execution.
  • No write actions in v1.

Commands

/dune health, /dune status, /dune readiness, and /dune services.
