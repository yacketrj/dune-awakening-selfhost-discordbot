# PR-0024: Promote v0.1.1

## Summary

Promotes the verified `v0.1.1-rc.1` release candidate to stable `v0.1.1`.
This updates package, lockfile, addon metadata, changelog, roadmap release
state, and stable release notes.

## User Impact

Operators get a stable patch release for the release-candidate workflow and
roadmap update. Runtime bot behavior is unchanged from `v0.1.0`.

## Security Impact

- Command surface: unchanged.
- RBAC or authorization: unchanged.
- Secret handling: unchanged.
- Data crossing Discord/bot/WebUI boundaries: unchanged.
- Network exposure: unchanged.
- Release posture: stable promotion follows a verified prerelease candidate and
  requires the same gates, SBOM, and checksum evidence.

## Least Privilege

Runtime least privilege is unchanged. The optional addon remains
zero-permission and packages under the stable patch version.

## Tests and Evidence

- `npm run check`
- `npm audit --audit-level=moderate`
- Semgrep
- Gitleaks
- Trivy filesystem
- Dependency review
- SBOM generation
- Docker build
- Trivy image
- `v0.1.1-rc.1` published artifact checksum verification

## Known Limitations

This release does not include live Discord guild or live WebUI adapter smoke
testing. Those remain operator deployment checks.
