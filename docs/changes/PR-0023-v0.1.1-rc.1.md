# PR-0023: Prepare v0.1.1-rc.1

## Summary

Prepares the `v0.1.1-rc.1` release candidate after PR #22 added release
candidate support and roadmap guidance. The candidate packages the RC workflow
change set for prerelease validation before stable promotion.

## User Impact

Operators get a GitHub prerelease candidate with addon and SBOM artifacts. Bot
runtime behavior is unchanged from `v0.1.0`.

## Security Impact

- Command surface: unchanged.
- RBAC or authorization: unchanged.
- Secret handling: unchanged.
- Data crossing Discord/bot/WebUI boundaries: unchanged.
- Network exposure: unchanged.
- Release posture: candidate artifacts use the same gates, SBOM, and checksum
  requirements as stable releases.

## Least Privilege

Runtime least privilege is unchanged. The optional addon remains
zero-permission and packages under the prerelease version.

## Tests and Evidence

- `npm run check`
- `npm audit --audit-level=moderate`
- Semgrep
- Gitleaks
- Trivy filesystem
- Dependency review
- SBOM generation
- Docker build
- Trivy image

## Known Limitations

This PR prepares and publishes a prerelease candidate only. Stable promotion to
`v0.1.1` requires a separate PR, clean gates, and published artifact checksum
verification for this candidate.
