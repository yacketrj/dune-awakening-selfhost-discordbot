# Release v0.1.1-rc.1

Release date: 2026-06-28

## Summary

Release candidate for the release-candidate workflow and roadmap update. This
prerelease validates the new RC path before promoting the same change set to a
stable patch release.

## Compatibility

- Bot repository: `yacketrj/dune-awakening-selfhost-discordbot`
- Upstream stable baseline:
  `Red-Blink/dune-awakening-selfhost-docker@1bb72c5`
- Upstream stable tag reviewed: `v1.3.37`
- Upstream release-candidate tag observed: `v1.3.38-rc.1` at `233aedf`
- Adapter boundary: read-only routes only

The upstream RC was checked for Discord adapter route impact. No Discord
adapter route files changed between `v1.3.37` and `v1.3.38-rc.1`, so this bot
keeps `v1.3.37` as the stable compatibility baseline.

## Changes Since v0.1.0

- Release validation accepts prerelease SemVer versions such as `0.1.1-rc.1`.
- Addon packaging accepts prerelease SemVer versions and produces matching
  prerelease artifact names.
- The release workflow marks prerelease version tags as GitHub prereleases.
- The roadmap now records RC-to-stable release discipline.
- The non-read-only roadmap now requires write-capable releases to start as
  release candidates before stable promotion.
- Upstream evidence now records the standalone Windows reference clone and the
  latest observed upstream RC separately from the stable baseline.

## Security Posture

- Runtime command behavior is unchanged.
- RBAC behavior is unchanged.
- The bot remains read-only.
- The optional addon remains zero-permission.
- Release notes and PR evidence continue to require redaction of secrets, PII,
  SteamIDs, FuncomIDs, private addresses, and real names.

## Verification

Release candidates must pass:

- `npm run check`
- `npm audit --audit-level=moderate`
- Semgrep
- Gitleaks history and worktree scans
- Trivy filesystem scan
- Docker image build
- Trivy image scan
- published artifact checksum verification

## Known Limitations

- This is a prerelease candidate, not the default stable operator release.
- No live Discord guild or live WebUI deployment test is bundled with the
  release; those remain operator smoke tests.
- No write-capable commands are included.
